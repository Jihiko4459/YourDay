Положения и условия использования приложения "Твой День"

Последнее обновление: 11.06.2025

1. Общие положения

1.1. Приложение "Твой День" (далее — "Приложение") — это многофункциональный ежедневник, предназначенный для планирования задач, отслеживания привычек, ведения дневника благодарности, управления финансами, мониторинга здоровья и других целей, связанных с организацией повседневной жизни.

1.2. Используя Приложение, вы соглашаетесь с настоящими Условиями. Если вы не согласны, прекратите использование.

1.3. Разработчик оставляет право изменять Условия, уведомляя пользователей через Приложение или официальный сайт.

2. Регистрация и аккаунт

2.1. Для доступа к полному функционалу требуется регистрация с указанием email и пароля.

2.2. Пользователь несет ответственность за сохранность своих учетных данных.

2.3. Минимальный возраст для использования Приложения — 6 лет.

3. Права и обязанности пользователя

3.1. Пользователь обязуется:

  • Не использовать Приложение для незаконных целей.
  • Не передавать свои учетные данные третьим лицам.
  • Не нарушать работу Приложения (включая попытки взлома или распространения вредоносного ПО).

3.2. Пользователь имеет право:

  • Создавать, редактировать и удалять свои данные (задачи, заметки, привычки и т.д.).
  • Отказаться от уведомлений в настройках профиля.

4. Данные и конфиденциальность

4.1. Локальные данные (например, незавершенные задачи) хранятся на устройстве пользователя.

4.2. Для синхронизации между устройствами данные могут передаваться на серверы разработчика с использованием шифрования.

5. Ограничения ответственности

5.1. Разработчик не гарантирует бесперебойную работу Приложения и не несет ответственности за:

  • Потерю данных из-за сбоев устройства или действий пользователя.
  • Косвенные убытки (например, пропущенные встречи из-за отсутствия напоминаний).

5.2. Приложение предоставляется "как есть".

6. Авторские права

6.1. Весь контент (дизайн, текст, функционал) принадлежит разработчику. Копирование запрещено.

Политика конфиденциальности приложения "Твой День"

Последнее обновление: 11.06.2025

1. Какие данные собираются

1.1. Персональные данные:

  • Email, ник, дата рождения (для регистрации).
  • Добровольно вводимые пользователем данные: задачи, привычки, заметки, измерения здоровья, финансовая аналитика.

1.2. Технические данные:

  • Тип устройства, версия ОС (для улучшения совместимости).
  • Анонимная статистика использования функций (через Google Analytics/Firebase).

2. Как используются данные

2.1. Для:

  • Предоставления доступа к аккаунту и синхронизации данных.
  • Анализа эффективности функций (например, процент выполненных задач).
  • Отправки уведомлений (если пользователь не отключил их).

2.2. Данные не передаются третьим лицам, кроме случаев, требуемых законом.

3. Хранение и защита

3.1. Локальные данные (не синхронизированные) удаляются при деинсталляции Приложения.

3.2. Синхронизированные данные хранятся на защищенных серверах с ограниченным доступом.

4. Сторонние сервисы

4.1. Приложение может использовать:

  • Google Fit/Apple Health (для интеграции данных о здоровье, только с согласия пользователя).
  • Платежные системы (для покупок премиум-функций, если такие есть).

5. Права пользователя

5.1. Вы можете:

  • Запросить удаление аккаунта и всех данных через поддержку.
  • Отказаться от сбора аналитики в настройках.
